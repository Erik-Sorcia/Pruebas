Módulo 1: ¿Qué es una Variable?

Definición Técnica: En programación, una variable es un espacio de almacenamiento reservado en la memoria principal del computador (RAM) asociado a un identificador (nombre), cuyo valor guardado puede ser modificado durante la ejecución del programa.

Explicación en lenguaje sencillo:

Imagina una variable como una **caja de zapatos** que tiene una etiqueta escrita con plumón afuera. El nombre de la etiqueta no cambia (por ejemplo, "caja_edad"), pero lo que guardas adentro del contenedor lo puedes cambiar las veces que quieras.

Ejemplos de analogías y equivalencias:

  • En Matemáticas / Álgebra: Cuando dices que 
    x = 5
    , estás usando una variable llamada "x" para guardar el número 5.
  • En los Videojuegos: La variable "puntuacion" aumenta de valor cada vez que destruyes un objetivo o recoges una moneda.

Laboratorio de Práctica

En Álgebra

x = 5

En Código (Java)

int x = 5;

En la Memoria RAM

[ Caja X ] guarda el valor: 5
